数字千分符

```js
var reg =/(\d{1,3})(?=(\d{3})+(?:$|\.))/g；
"123564737.66".replace(reg,'$1,');
```

金额格式，仅允许输入数字和最多一位小数点，两位小数

```js
var reg = ;/^(0(\.|\.\d{1,2})?|[1-9]\d*(\.|\.\d{1,2})?)$/;
```

数字格式(包含小数)

```js
// 只有0开头，如果有小数，后面仅有1个点 点后面可以有多位数字
//或者
//1-9开头，1位或者多位整数，如果有小数，后面仅有1个点，点后面可以有多位数字
var reg = /^(0(\.|\.\d+)?|[1-9]\d*(\.|\.\d+)?)$/;
```
