栈的原则：先进后出

通过push和pop实现

```js
class Stack{
  constructor(){//初始化一个空的栈
    this.items = [];
  }
  push(item){//入栈
    this.items.push(item);
  }
  pop(){//出栈
    return this.items.pop();
  }
  peek(){//查看栈顶元素
    return this.items[this.items.length-1];
  }
  size(){//查看栈的大小
    return this.items.length;
  }
  isEmpty(){//查看栈是否为空
    return this.items.length === 0;
  }
  clear(){//清空栈
    this.items = [];
  }
  print(){//查看栈中所有内容
    console.log(this.items.toString())
  }
}
```

利用栈实现十进制转为二进制：“除2取余，逆序排列”

```js
function divideBy2(num){
  var str = '',stack = new Stack();
  while(num){
    stack.push(num%2);//求余入栈
    num = Math.floor(num/2);//原数除2
  }
  while(!stack.isEmpty()){
    str += stack.pop().toString();//出栈，拼接字符串
  }
  return str;
}
console.log(divideBy2(10));//1010
```
